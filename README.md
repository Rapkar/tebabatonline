## Description

**TCRM** is a comprehensive Laravel-based Customer Relationship Management (CRM) system designed to streamline customer interactions and business processes. It features real-time chat, ticket management, order tracking, and data visualization tools for shop counts and post views. TCRM empowers businesses to enhance customer relationships, improve operational efficiency, and gain actionable insights.

## Features

- **Order Management**: Track and manage customer orders efficiently.
- **Real-Time Chat**: Communicate with customers instantly through an integrated chat system.
- **Ticketing System**: Handle customer support with a robust ticket management feature.
- **Analytics Dashboard**:
  - Visualize shop counts with dynamic charts.
  - Monitor post views for content engagement insights.
- **User-Friendly Interface**: Intuitive design for seamless navigation and usability.
- **Customizable Workflows**: Adaptable to various business needs.

## Technologies Used

- Laravel (v10.x)
- PHP (v8.1 or higher)
- MySQL/MariaDB
- Real-Time Broadcasting (e.g., Pusher or Laravel Echo)
- Chart.js (or similar library for data visualization)
